A guide for expanding your knowledge

The purpose of this page is to serve as the next step if you wish to continue to learn about the technologies used in this project. We will be providing sources you can use to further your knowledge on web scraping and machine learning. However, our primary focus will be on the enormous field of artificial intelligence and it's subsets of machine learning and deep learning (the latter currently being the primary focus of data scientists and researchers in the field).

The first references will be about web scraping. We recommend you take a look at these if you wish to implement web scraping into your own project or simply want to know more about the process. Following that will be references for expanding your knowledge on shallow machine learning, the type of machine learning we used in this project. However, also provide references for deep learning (an incredibly interesting yet more difficult subfield of machine learning).
