A Web App

The purpose of this website is to give users insight into our project. Our primary aim is to help you understand the concepts behind seemingly difficult technologies that make use of ideas such as machine learning. However, we also wanted to bring to life a product that would benefit others. To this end, we developed a web scraper that allows us to collect information from the internet as well as tools that help us analyze this information and find patterns behind it. In the previous page, we showed you how some of those tools look and how they could benefit us. However, those were static representations that only applied to one dataset. We want our web scraper and analytical tools to be dynamic and apply to a wide vairety of datasets.

To make this a reality, we have developed a dynamic web application. We encourage you to visit this web app yourself and be able to benefit from it. If there is something you find that is difficult to understand, please return here and use this site (or one of the many referenced sites) as a guide. What follows is a minimized window of the web app that you can use.

What's Next?

In essence, this project is simply one big idea: Everyone can and should understand artificial intellignce (ai) and machine learning. We sincerely hope you have been able to benefit from this project and wish that you are able to continue on this journey of learning. We encourage you to build upon this knowledge and find ways to implement what you learn into your own profession, knowing that as machine learning and ai become more and more prevalent in our everyday lives, being prepared for the changes is of essence. May the information provided in these pages serve as a starting point.

If you would like to build on our work, feel free to visit our public GitHub repository where you will be able to download the source code for the web scraper, our web app, and even this very website. Additionally, full documentation on all these elements will be available, as well as videos further explaining how they work.

We do not wish to leave you alone if you choose to continue to learn about these technolgies. To help point you in the right direction, visit our "More Info" page. A link to this page is included in the navigation header. In this guide you will be able to find a variety of references that will serve as tools for continuing to expand your knowledge.
